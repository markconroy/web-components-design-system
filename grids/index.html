Grids

Grid: 12 Columns

This is a very simple grid system, based loosely on the GOV.UK grid system.

The number of columns be changed via the --grid-columns variable in CSS.

You can change the space between items via the --grid-gap-* variables.

Grid: No Gap Between Items

.mc-grid .mc-grid--gap-none

Grid: Small Gap Between Items

.mc-grid .mc-grid--gap-small

Grid: Large Gap Between Items

.mc-grid .mc-grid--gap-large
