The basic container is a custom element called mc-container, which evaluates to the a max-width of --mc-width-content-area. It also has margin-left and margin-right set to auto, so it aligns in the centre of the screen if the screen is wider than its max-width.

It accepts props for padding, size, gap and alignment, with the following allowed values:

  • Padding:
    • inline - inline-padding is set to --mc-spacing
    • block - block-padding is set to --mc-spacing
    • all - padding is set to --mc-spacing
  • Size:
    • small - max-width is set to --mc-width-small
    • medium - max-width is set to --mc-width-medium
    • large - max-width is set to --mc-width-large
    • xlarge - max-width is set to --mc-width-xlarge
    • 100 - max-width is set to 100%
  • Gap:
    • small - gap is set to --mc-spacing-small
    • medium - gap is set to --mc-spacing-medium
    • large - gap is set to --mc-spacing-large
    • larger - gap is set to --mc-spacing-larger
    • largest - gap is set to --mc-spacing-largest
  • Alignment:
    • start - aligns the container to the inline-block-start
    • end - aligns the container to the inline-block-end

If you use the gap property, the container sets display: flex on itself with a one column grid, then each item placed in it will have the specified gap between itself and the next item. This is handy if you wanted to place two cards on top of each other but not have them touching off each other.

        <mc-container>
