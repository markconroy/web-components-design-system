Containers

The basic container has a class of mc-container, which evaluates to the a max-width of --width-content-area. It also has margin-left and margin-right set to auto, so it aligns in the centre of the screen if the screen is wider than its max-width.

All other containers are modifiers of this one, which allows them to be wider or narrower. Their widths are set by a corresponding modifier class such as mc-container--small or mc-container--xlarge for example.

If you want to align your layout to the left or right - often needed to have, for example, a text area aligned to the left that is only X space wide, you can add a class of mc-container--left or mc-container--right.

Container: Default

        
        <div class="mc-container">
          HTML here
        </div>
