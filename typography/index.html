All of our fonts are responsive. They go from a minimum size on small screens to a maximum size on large screens. In between these screen sizes they grow or shrink until they reach one of these extremes.

We do not set font sizes necessarily by elements. For example, not all <h2> elements are the same size: a <h2> used in a title for search results may be a different size to a <h2> used as a title for a menu in the footer. Instead, we create variables for all the combinations of font sizes that are in the design, and apply these where needed.

This approach means we can have as many font sizes as we need, and can apply them where they are needed. So ... we can use <h2> where it is needed semantically, but another class for the visual style of it.
