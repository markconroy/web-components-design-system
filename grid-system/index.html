This is a very simple grid system, based on a 12-column grid.

It has two custom elements available:

  1. A <mc-grid> element
  2. A <mc-grid-item> element

The grid container accepts two attributes, gap and size. Gap can be set to small, medium, large, or none. If this attribute is left out, it will default to medium, which equates to --mc-spacing. Size can be set to quarters, thirds, or halves. This makes it very handy for a lot of use cases, however if you want different widths per item, you can use the <mc-grid-item> custom element and set the size you want on that element.

The grid items accept one attribute, size. This can be set to one-quarter, one-third, one-half, two-thirds, or three-quarters. If this attribute is left out, it will default to one-third.
